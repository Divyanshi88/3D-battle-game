# 🎮 3D Battle Game

Welcome to **3D Battle Game** – a simple yet exciting game where you can control characters and engage in thrilling battles in a 3D environment.

---

## 🚀 Project Overview

This project is a 3D battle game where:
- Players can select their characters.
- Characters can engage in battles.
- Currently working on fixing player animations and visibility issues.

---

## 🎯 Features
- 🔸 3D Characters and Environment
- 🔸 Character Selection
- 🔸 Basic Battle Setup
- 🔸 Custom Camera Setup
- 🔸 Game Scene with User Controls

---
